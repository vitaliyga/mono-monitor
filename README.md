# mono-rate-bot

Telegram-бот на Cloudflare Workers. Следит за разницей (спредом) между курсом
**продажи** и **покупки** доллара и евро в monobank и присылает уведомление,
когда спред опускается до заданного порога.

Спред = `rateSell − rateBuy` — это ровно то, во что обходится «продать и сразу
купить»: чем он меньше, тем дешевле круговая сделка.

## Команды бота

| Команда         | Что делает                                                |
|-----------------|-----------------------------------------------------------|
| `/start`        | Включить уведомления                                      |
| `/set usd 0.2`  | Порог разницы для доллара (грн)                           |
| `/set eur 0.3`  | Порог разницы для евро                                    |
| `/set 0.25`     | Один порог сразу для обеих валют                          |
| `/status`       | Курс, спред, порог и минимум спреда по обеим валютам      |
| `/check`        | Текущий курс прямо сейчас                                 |
| `/reset`        | Обнулить запомненный минимум спреда                       |
| `/stop`         | Выключить уведомления                                     |
| `/help`         | Справка                                                   |

Когда `спред ≤ порога`, бот пишет один раз по валюте. Повторно сработает только
после того, как спред снова вырастет выше порога. Минимум спреда запоминается с
момента последнего `/reset` — удобно, чтобы понять, насколько низко он опускается,
и выставить реалистичный порог.

## Деплой

Нужен Node.js и установленный wrangler (`npm i -g wrangler` или через `npx`).

### 1. Создать бота
Напиши [@BotFather](https://t.me/BotFather) → `/newbot` → получи токен.

### 2. Поставить зависимости и войти в Cloudflare
```bash
npm install
npx wrangler login
```

### 3. Создать KV-хранилище
```bash
npx wrangler kv namespace create KV
```
Скопируй полученный `id` в `wrangler.toml` (поле `id` в блоке `kv_namespaces`).

### 4. Добавить секреты
```bash
npx wrangler secret put BOT_TOKEN
# вставь токен от BotFather

# любой случайный набор символов — защита вебхука
npx wrangler secret put WEBHOOK_SECRET
```

### 5. Задеплоить
```bash
npx wrangler deploy
```
Получишь адрес вида `https://mono-rate-bot.<твой-аккаунт>.workers.dev`.

### 6. Привязать вебхук Telegram
Подставь свой токен, адрес воркера и тот же WEBHOOK_SECRET:
```bash
curl "https://api.telegram.org/bot<BOT_TOKEN>/setWebhook?url=https://mono-rate-bot.<account>.workers.dev&secret_token=<WEBHOOK_SECRET>"
```

Готово. Напиши боту `/start`.

## Локальная проверка cron
```bash
npx wrangler dev
# затем дернуть scheduled-обработчик:
curl "http://localhost:8787/__scheduled?cron=*/10+*+*+*+*"
```

## Заметки
- Наличный курс USD/UAH у monobank по ночам/выходным может отсутствовать —
  тогда `/check` вернёт «курс недоступен», это нормально.
- Частоту проверки меняй в `wrangler.toml` (`crons`). Чаще ~5 минут смысла нет:
  monobank кеширует курс и лимитирует запросы.
