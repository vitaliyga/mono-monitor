# mono-rate-bot

Telegram-бот на Cloudflare Workers. Стежить за різницею (спредом) між курсом
**продажу** і **купівлі** долара та євро в monobank і надсилає сповіщення,
коли спред опускається до заданого порога.

Спред = `rateSell − rateBuy` — це саме те, у що обходиться «продати й одразу
купити»: що він менший, то дешевша кругова угода.

## Команди бота

| Команда         | Що робить                                                 |
|-----------------|-----------------------------------------------------------|
| `/start`        | Увімкнути сповіщення                                      |
| `/set usd 0.2`  | Поріг різниці для долара (грн)                            |
| `/set eur 0.3`  | Поріг різниці для євро                                    |
| `/set 0.25`     | Один поріг одразу для обох валют                          |
| `/status`       | Курс, спред, поріг і мінімум спреду по обох валютах       |
| `/check`        | Поточний курс просто зараз                                |
| `/reset`        | Обнулити запам'ятований мінімум спреду                    |
| `/stop`         | Вимкнути сповіщення                                       |
| `/help`         | Довідка                                                   |

Коли `спред ≤ порога`, бот пише один раз по валюті. Повторно спрацює лише
після того, як спред знову зросте вище порога. Мінімум спреду запам'ятовується з
моменту останнього `/reset` — зручно, щоб зрозуміти, наскільки низько він опускається,
і виставити реалістичний поріг.

## Деплой

Потрібен Node.js і встановлений wrangler (`npm i -g wrangler` або через `npx`).

### 1. Створити бота
Напиши [@BotFather](https://t.me/BotFather) → `/newbot` → отримай токен.

### 2. Поставити залежності та увійти в Cloudflare
```bash
npm install
npx wrangler login
```

### 3. Створити KV-сховище
```bash
npx wrangler kv namespace create KV
```
Скопіюй отриманий `id` у `wrangler.toml` (поле `id` у блоці `kv_namespaces`).

### 4. Додати секрети
```bash
npx wrangler secret put BOT_TOKEN
# встав токен від BotFather

# будь-який випадковий набір символів — захист вебхука
npx wrangler secret put WEBHOOK_SECRET
```

### 5. Задеплоїти
```bash
npx wrangler deploy
```
Отримаєш адресу на кшталт `https://mono-rate-bot.<твій-акаунт>.workers.dev`.

### 6. Прив'язати вебхук Telegram
Підстав свій токен, адресу воркера й той самий WEBHOOK_SECRET:
```bash
curl "https://api.telegram.org/bot<BOT_TOKEN>/setWebhook?url=https://mono-rate-bot.<account>.workers.dev&secret_token=<WEBHOOK_SECRET>"
```

Готово. Напиши боту `/start`.

## Локальна перевірка cron
```bash
npx wrangler dev
# потім смикнути scheduled-обробник:
curl "http://localhost:8787/__scheduled?cron=*/10+*+*+*+*"
```

## Нотатки
- Готівковий курс USD/UAH у monobank уночі/у вихідні може бути відсутній —
  тоді `/check` поверне «курс недоступний», це нормально.
- Частоту перевірки змінюй у `wrangler.toml` (`crons`). Частіше ніж ~5 хвилин сенсу немає:
  monobank кешує курс і лімітує запити.
